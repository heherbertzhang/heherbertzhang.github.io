UoftHacks
Homey

Homey

Project Description

Homey is a home monitoring system which is powered by Intel Edison and can be controlled using Android and Pebble watches. This project won the best use of Intel Technology in UofTHacks 2015. Homey could receive instructions from multiple android devices or pebble devices configured to the Intel Edison. It can measure temperature and display temperature, check if the lights are on detect intruders when you are away, saving electricity and protecting your home while you are away.


  • Designed a Home Monitoring System using Intel Edison Platform
  • Used JavaSctipt collect data from various sensors
  • Designed a companion Pebble Smart Watch App that received the data from sensors using AJAX
  • Effectively used NodeJS with Express Framework to transmit the sensor data to the companion Android and Pebble app

Project Source GitHub.

Top Hack

Top Hack

Project Description

Top Hack is an Android Application which automates answering of top hat (A classroom management tool) questions. Top Hack exploits the off-line mode available in the Top Hat suite to send answers to the top hat servers remotely. Once enabled, the users friend sends the code and answer to the users phone. The Top Hack app then recognises the message containing the code and answer and sends it to the top hat server with the users credentials.

  • Can differentiate between top hat codes and normal messages.
  • Establishes connection to top hat server and displays status or update.
  • Meme based theme for younger generations who will use this app.

The app can be found at xda link.
